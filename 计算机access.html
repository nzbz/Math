  3. 关系数据库

    关系:一个关系就是一张二维表，每个关系有一一个关系名。在Access中，一个关系就是数据库文件中的一个表对象。 ​ 属性:二维表中垂直方向的列称为属性，有时也叫做一个字段 ​ 域:-一个属性的取值范围叫做一个域。 ​ 元组:二维表中水平方向的行称为元组，有时也叫做一条记录。

    码:又称为关键字。二维表中的某个属性或属性组，若它的值唯一地标识了一个元组，则称该属性或属性组为候选码。若一个关系有多个候选码，则选定其中一个为主码，也称为主键(不能为空、不能重复) ​ 分量:元组中的一个属性值叫做元组的一个分量。 ​ 关系模式:是对关系的描述，它包括关系名、组成该关系的属性名、属性到域的映象。通常简记为:关系名(属性名1，属性名2，.... 属性名n)。

  4. 关系运算

    有两种:

    1. 传统的集合运算(并、差、交、广义笛卡儿积等);

      1. 专门的关系运算(选择、投影、连接)。
      • 选择: 在关系中选择满足指定条件的元组(行)。
      • 投影: 在关系中选择某些属性(列)

  2. Access

    Access 2010关系数据库是数据库对象的集合。

    数据库对象包括表、查询(第一层次)、 窗体、报表(第二层次)、宏、模块(第三层次)。 Access只能打开并运行一个数据库。但是，在每一个数据库中，可以拥有众多的表、查询、窗体、报表、宏和模块。这些数据库对象都存储在同一个以.accdb为扩展名的数据库文件中

    表对象:

    1 表是有结构的数据的集合，表用于存储基本数据，是其他数据库对象的基础, 在表中数据是按行和 列存储的，相当于由行和列组成的二维表格。在表中，一行数据称为一条记录,一列称为一个字段。 ​ 2 一个表 是由表结构和表内容构成。表有两个视图: ​ 设计视图(结构) 和 数据表视图(内容)

    表结构:

    字段名称: 长度<=64 不能包括 . ! [ ] ’ 不能空格开头

    数据类型: 12种 文本(最长255个字符)、备注(长文本)、数字、日期/时间、货币、自动编号、是/否、OLE对象(word、Excel、图片、声音等)、超链接、附件、计算、查阅向导

    查询对象:

    查询主要有:

    (1)选择查询。选择查询是用于从表中检索数据或进行计算的查询。这也是最常用的查询类型。 ​ (2)生成表查询。生成表查询就是通过查询来搜索记录，然后将这些记录保存到一个新的表中。 ​ (3)追加查询。追加查询可以将一个数据表中的数据添加到另一个数据表中

    (4)更新查询。更新查询就是利用查询的功能批量更改记录。 ​ (5)交叉表查询。交叉表查询主要用于显示某一个字段数据的统计值，比如计数、平均值等， ​ (6)删除查询。删除查询就是通过查询来搜索符合条件的记录，然后运行该查询将这些记录从原数据表中 删除

    窗体对象Form:

    窗体是应用程序和用户之间的接口界面，为用户查看和编辑数据库中的数据提供了一种友好的交互式界面。使用窗口的形式显示给用户，用户可以创建、修改、删除数据

    报表(Report)对象 报表是以打印格式显示用户数据的一种有效方式。用户只能通过报表浏览数据，而不能编辑数据。

  1. 查询分类

    • 选择查询

      1. 单表查询
      2. 连接查询

    • 操作查询

      操作查询 包括:更新查询、生成表查询、追加查询、删除查询

    • SQL查询

      基本格式:SELECT表名[INTO目标表]FROM表名[WHERE条件][ ORDER BY字段][ GROUP BY字段[ HAVING条件]] (1)ORDER BY字段:按指定字段排序; asc代表的是升序，desc代表 的是降序如果不写，默认是升序(2)GROUP BY字段:按指定字段分组; (3)HAVING条件:设置分组条件; (4)INTO目标表:将查询结果输出到指定的目标表。 示例:查询sd表中女同学的信息，并将查询结果 输出到“女生”表。 SELECT* INTO女生 FROM sd WHERE 性别=“女”

      UPDATE语句(字段内容更新)

      基本格式: UPDATE表名SET字段=表达式[ WHERE条件] '示例: 将班级编号为“201001" 的记录的班级编号修改为“201010 UPDATE sd SET 班级编号=“201010”WHERE 班级编号 =“201001"

      INSERT语句(插入记录)

      基本格式:INSERTINTO 表名 (字段名表) VALUES (内容列表) 示例:在sd表中插入一条记录。 INSERT INTO sd(学号,姓名，性别,班级编号) VALUES("201001011","张山”,“女","201001")

      DELETE语句(删除记录)

      基本格式: DELETE FROM表名[ WHERE条件] 示例:删除sd表中班级编号为“201001”的所有记录。 DELETE FROM sd WHERE 班级编号 =“201001"

多媒体技术

媒体两种含义

一是 指存储信息的实体，例如磁盘、磁带、光盘、半导体存储器等: 二是 指传递信息的载体，例如声音、 图像、图形、文字等。

多媒体技术

多媒体技术:把数字、文字、声音、图形、图像及动画等各种媒体有机组 合起来，利用计算机、通信和广播技术，使它们建立逻辑联系，并能进行加工处理的技术。

媒体分类

  • 表示媒体: 为了传送感觉媒体而人为研究出来的媒体，一般以编码的形式描述，如静止图像

  • 感觉媒体: 直接作用于人的感觉器官，如声音、文字

  • 显示(呈现、表现)媒体: 进行信息输入和输出的媒体，如 键盘 鼠标

  • 存储媒体: 存放感觉媒体数字化后的代码的媒体 如 磁带、光盘

  • 传输媒体: 传输信息的物理设备，如电话线、电缆

图像处理

图形图像区别

图形称矢量图形或几何图形，用数学方法来表示图形。放大后不失真，所占存储空间小。

图像又称点阵图像或位图图像，是空间和亮度上已经离散化的图像。是一个矩形，矩形中的任一元素都对应图像上的一个点，每个点都有它的灰度或颜色等级。放大后失真，所占存储空间大。

图像文件的字节数= 图像分辨率 * 颜色深度 / 8

1字节(Byte） = 8位(bit)
